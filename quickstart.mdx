title: "Quickstart: Download, Configure, and Chat with CodePilot"

---

CodePilot offers two ways to get started: download a pre-built release (recommended for most users), or build from source if you want to contribute or run the latest code. Either way, you'll be in a conversation with an AI agent in just a few minutes.

<Tabs>
  <Tab title="Path A: Download a release">
    <Steps>
      <Step title="Download the installer">
        Visit the [Releases page](https://github.com/op7418/CodePilot/releases) and download the installer for your platform:

        | Platform | Download | Architecture |
        |---|---|---|
        | macOS | Apple Silicon (.dmg) · Intel (.dmg) | arm64 / x64 |
        | Windows | Installer (.exe) | x64 + arm64 |
        | Linux | AppImage · .deb · .rpm | x64 + arm64 |
      </Step>
      <Step title="Install and launch">
        Run the installer and open CodePilot. On macOS, Gatekeeper may show a warning on first launch — see the [Installation guide](/installation) for workarounds.
      </Step>
      <Step title="Configure a provider">
        Go to **Settings → Providers** and add credentials for the provider you want to use. CodePilot includes presets for all major providers — pick one and enter your API key.

        <Tip>
          Don't have an API key yet? [OpenRouter](https://openrouter.ai) gives you access to many models through a single key with pay-as-you-go pricing.
        </Tip>
      </Step>
      <Step title="Start a conversation">
        Click **New Chat** in the sidebar. Choose a working directory, select a mode (**Code**, **Plan**, or **Ask**), pick a model, and start chatting.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Path B: Build from source">
    <Steps>
      <Step title="Install prerequisites">
        You need Node.js 18+ and npm 9+ (ships with Node 18).

        | Prerequisite | Minimum version |
        |---|---|
        | Node.js | 18+ |
        | npm | 9+ |
      </Step>
      <Step title="Clone and install">
        ```bash
        git clone https://github.com/op7418/CodePilot.git
        cd CodePilot
        npm install
        ```
      </Step>
      <Step title="Run in development mode">
        <CodeGroup>
          ```bash Browser mode
          npm run dev
          # Opens at http://localhost:3000
          ```
          ```bash Full desktop app
          npm run electron:dev
          # Starts Next.js + Electron shell
          ```
        </CodeGroup>

        Use `npm run dev` for quick iteration in the browser. Use `npm run electron:dev` for the full desktop experience with native window controls and file system access.
      </Step>
      <Step title="Configure a provider">
        Open **Settings → Providers** and add your API key for any supported provider.
      </Step>
    </Steps>
  </Tab>
</Tabs>

## First launch checklist

Once CodePilot is running, work through these setup steps:

<Steps>
  <Step title="Configure a provider">
    Go to **Settings → Providers** and add credentials for the provider you want to use. Run **Diagnostics** to verify connectivity.
  </Step>
  <Step title="Create your first conversation">
    Click **New Chat**. Pick a working directory (the folder your assistant will work in), select a mode, choose a model, and send your first message.
  </Step>
  <Step title="Set up Assistant Workspace (optional)">
    Go to **Settings → Assistant**, choose a workspace directory, and enable **Onboarding**. CodePilot will create `soul.md`, `user.md`, `claude.md`, and `memory.md` at the workspace root and walk you through a 13-question onboarding flow.

    | File | Purpose |
    |---|---|
    | `soul.md` | AI personality and communication style |
    | `user.md` | Your profile and preferences |
    | `claude.md` | Rules and constraints for the AI |
    | `memory.md` | Long-term notes (append-only) |
  </Step>
  <Step title="Add MCP servers and skills (optional)">
    Navigate to the **MCP** page in the sidebar to add Model Context Protocol servers that give your assistant access to external tools and data sources. Visit the **Skills** page to install reusable agent behaviors from the [skills.sh](https://skills.sh) marketplace.
  </Step>
</Steps>

<Note>
  Installing the [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code/overview) (`npm install -g @anthropic-ai/claude-code`) unlocks advanced capabilities like direct file editing, terminal commands, and git operations. It is recommended for Code mode but not required for basic chat with any provider.
</Note>
