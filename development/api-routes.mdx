CodePilot exposes an internal REST API implemented as Next.js App Router route handlers under `src/app/api/`. These endpoints are consumed exclusively by the Electron-hosted frontend — they are not a public API and have no versioning or authentication. All routes run on the Node.js runtime and have direct access to the SQLite database and filesystem.

<Note>
These are internal APIs. They are not designed for external consumption and may change between releases without notice.
</Note>

## Route groups

### Chat (`/api/chat`)

The main AI interaction surface. The root `POST /api/chat` is the most important endpoint in the codebase — it drives all AI streaming.

| Method | Path | Description |
|---|---|---|
| `POST` | `/api/chat` | Send a message and stream the Claude Agent SDK response via SSE |
| `GET/POST/DELETE` | `/api/chat/sessions` | List, create, or manage chat sessions |
| `GET/PATCH/DELETE` | `/api/chat/sessions/[id]` | Read, update, or delete a specific session |
| `GET` | `/api/chat/sessions/by-cwd` | Look up the most recent session for a working directory |
| `POST` | `/api/chat/messages` | Persist a message to the DB without triggering the model |
| `PUT` | `/api/chat/messages` | Update an existing message's content |
| `POST` | `/api/chat/interrupt` | Abort the active stream for a session |
| `POST` | `/api/chat/rewind` | Rewind a session to a previous checkpoint |
| `GET/PUT` | `/api/chat/mode` | Get or set the interaction mode for a session |
| `GET/PUT` | `/api/chat/model` | Get or set the model for a session |
| `GET/POST` | `/api/chat/permission` | Read or resolve a pending tool permission request |
| `POST` | `/api/chat/structured` | Request a structured (non-streaming) AI response |

#### Example: `POST /api/chat/messages`

Persists a message directly to the database. Used by image-gen mode to write messages without invoking the model.

**Request body:**

```json
{
  "session_id": "a1b2c3d4e5f6...",
  "role": "user",
  "content": "[{\"type\":\"text\",\"text\":\"Generate a landscape image\"}]",
  "token_usage": null
}
```

**Success response (200):**

```json
{
  "message": {
    "id": "f7e8d9c0b1a2...",
    "session_id": "a1b2c3d4e5f6...",
    "role": "user",
    "content": "[{\"type\":\"text\",\"text\":\"Generate a landscape image\"}]",
    "token_usage": null,
    "created_at": "2026-04-21 12:34:56"
  }
}
```

**Error responses:** `400` if required fields are missing, `404` if the session does not exist.
