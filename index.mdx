CodePilot is a desktop AI agent client built with Electron and Next.js that gives you a single interface across 17+ AI providers — from Anthropic and OpenRouter to AWS Bedrock, Google Vertex, Ollama, and custom endpoints. Every conversation, setting, and memory stays on your machine in a local SQLite database.

## Get up and running

<Steps>
  <Step title="Download CodePilot">
    Visit the [Releases page](https://github.com/op7418/CodePilot/releases) and download the installer for your platform — macOS (Apple Silicon or Intel), Windows, or Linux.
  </Step>
  <Step title="Configure a provider">
    Open **Settings → Providers** and add your API key for any supported provider. CodePilot includes presets for all major providers — just pick one and enter your key.
  </Step>
  <Step title="Start a conversation">
    Click **New Chat**, pick a working directory, choose a mode (Code / Plan / Ask), select a model, and start talking to your AI assistant.
  </Step>
  <Step title="Extend with MCP and skills">
    Add MCP servers via the **MCP** page in the sidebar to give your assistant access to tools, databases, and APIs. Install reusable skills from the skills.sh marketplace on the **Skills** page.
  </Step>
</Steps>

<Note>
  Installing the [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code/overview) (`npm install -g @anthropic-ai/claude-code`) unlocks file editing, terminal commands, and git operations. It is recommended but not required — you can use any supported provider for chat without it.
</Note>
